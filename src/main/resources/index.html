MemMunk is a live LevelDB-style key-value store built from scratch in Java — memtable, write-ahead log, SSTables, and compaction, all running in production.
put
Write any key-value pair. Writes go to the in-memory memtable first. Keys and values can be any string or JSON.
get / delete
Look up or tombstone a key. Deletes write a marker that is cleaned up during compaction.
scan by prefix
Returns all live keys starting with a given string. Try user: to see all user keys. Fast because keys are sorted.
flush
Push the memtable to disk as an immutable SSTable file. Happens automatically at the size threshold.
compact
Merge SSTable files within the same size tier, reclaim space, and purge deleted keys.
stats row
Live memtable entry count, SSTable file count, total keys across all layers, and active compaction tiers.
